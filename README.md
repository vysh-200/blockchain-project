# 🔗 Blockchain Project

This is a simple blockchain project built using Python and Flask.

---

## 📌 What this project does
- Creates blocks
- Links blocks using hashes
- Adds transactions
- Validates blockchain

---

## ⚙️ How it works
1. A block is created
2. Each block connects to previous block
3. Proof of Work is used to mine blocks

---

## 🚀 How to run
pip install -r requirements.txt  
python main.py
